    + [london]
    + [newyork]
    + [sanfrancisco]
    + [tokyo]

    + [london (伦敦)]
    + [newyork (纽约)]
    + [sanfrancisco (旧金山)]

## 使用

Zeal 内建的下载器经常下载缓慢、下载失败，可以尝试手动从离自己最近的位置下载，
并解压至文档存放目录 (默认位置: `$HOME/.local/share/Zeal/Zeal/docsets`)。
