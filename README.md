## 使用

如遇 Zeal 内建的下载器经常下载缓慢、下载失败，可以尝试手动从离自己最近的服务器下载，
并解压至文档存放目录。

文档默认位置: `$HOME/.local/share/Zeal/Zeal/docsets`。
